Q1What is the document object model(DOM)? ▼

A1: The Document Object Model (DOM) is a cross-platform and language-independent API that treats an HTML or XML document as a tree structure, where each node represents a part of the document, such as an element, attribute, or piece of text.

Q2How do you select an element from the dom ? ▼

A2: getElementById(): This method selects a single element by its unique id attribute. getElementsByTagName(): This method selects all elements with a specified tag name. getElementsByClassName(): This method selects all elements that have a specified class. getElementsByName(): This method selects all elements that have a specified name attribute . querySelectorAll(): This method selects all elements that match a specified CSS selector.

Q2What is event deligation in the context of the DOM?And why is usefull? ? ▼

A2: Event delegation in the context of the DOM is a technique where a single event listener is attached to a parent element to handle events for its child elements.
